## 📖 About

Choosing the next book to read can be overwhelming.

**Book Buddy AI** solves this problem by providing intelligent, personalized book recommendations using Artificial Intelligence.

Whether you're a beginner building a reading habit or an avid reader exploring new genres, Book Buddy AI helps you discover books you'll love.

## ✨ Features

- 🤖 AI-powered book recommendations

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/book-buddy-ai.git
```

### Navigate to Project

```bash
cd book-buddy-ai
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file.

```env
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=

MONGODB_URI=

OPENAI_API_KEY=

NEXTAUTH_SECRET=
```

### Run Development Server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```
